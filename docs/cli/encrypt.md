# Commande `encrypt`

La commande `encrypt` permet de chiffrer une archive `.oxz` existante avec un mot de passe.

**Exemple d'utilisation :**
```bash
oxide encrypt archive.oxz
```

## Arguments et Options

| Option / Argument | Description | Valeur par défaut |
| --- | --- | --- |
| `<input>` | L'archive `.oxz` source à chiffrer. (Requis) | *Aucun* |
| `-o, --output` | Le chemin de destination. | Remplace le fichier original en toute sécurité |
