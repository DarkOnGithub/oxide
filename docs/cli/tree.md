# Commande `tree`

Affiche l'architecture et le contenu d'une archive `.oxz`. Très utile pour comprendre de quoi est faite une archive avant d'en lancer l'extraction complète.

**Exemple d'utilisation :**
```bash
oxide tree archive.oxz
```

## Arguments et Options

| Option / Argument | Description | Valeur par défaut |
| --- | --- | --- |
| `<input>` | Le fichier de l'archive `.oxz` à analyser. (Requis) | *Aucune* |
